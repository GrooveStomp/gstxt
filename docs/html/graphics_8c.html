uint32_t AlphaBlendPixels ( uint32_t  top,
uint32_t  bottom 
)

compute the ADD OVERLAY alpha blend of top over bottom

top is a 32-bit (R|G|B|A) color and bottom is also a 32-bit (R|G|B|A) color. We compute the ADD OVERLAY operation of top overlayed onto bottom.

To achieve this, we splite each 32-bit integer into its corresponding R,G,B,A components and then invoke AlphaBlendComponent() on each R,G,B component.

Parameters
[in]top32-bit color (R|G|B|A) to overlay
[in]bottom32-bit color (R|G|B|A) to be overlaid upon
Returns
32-bit color (R|G|B|A) result of the ADD OVERLAY operation
See also
https://en.wikipedia.org/wiki/Alpha_compositing#Description
